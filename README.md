# API


## login

method: POST

url: /login

arguments(form data): phone, password

return: user dict with name, id, content and so on.


## logout

method: PUT

url: /logout

arguments: None

return: 'Logout Success!' 200


## signup

method POST

url: /signup

arguments(form data): phone(required), password(required), name

return: user dict...
